# sortpy
sortpy is a tool to visualize common sorting algorithms with ease. <br />
Requires graphics.py. <br />
<br />
![](https://media.giphy.com/media/oOLkPK0ARpO3ajaqOB/giphy.gif)

## Supported algorithms

### Bubble sorts
* Bubble sort
* Cocktail sort
* Comb sort
* Odd-even sort

### Insertion sorts
* Insertion sort
* Shellsort
* Gnome sort

### Divide and conquer
* Merge sort
* Quicksort

### Other
* Selection sort
* Stooge sort
